# ToDo App (C# Console)

Eine einfache **ToDo-Anwendung in C#** für die Konsole.  
Das Projekt dient zum Lernen der Grundlagen von C# und sauberer Programmstruktur.

---

## Funktionen

- Aufgabe hinzufügen
- Aufgaben anzeigen
- Aufgabe löschen (per Nummer)
- Alle Aufgaben löschen (mit Sicherheitsabfrage)
- Sichere Eingaben mit `TryParse`
- Menüführung in der Konsole

---

## Gelernte Konzepte

- C# Grundlagen
- `List<string>`
- Methoden (`void`, Rückgabewerte)
- `switch` / `while`
- Benutzereingaben (`Console.ReadLine`)
- Fehlervermeidung mit `TryParse`
- Strukturierter Code

---
